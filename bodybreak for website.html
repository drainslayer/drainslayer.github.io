    • Lists:

    There are two types of lists in this element. Ordered and Unordered lists.

    Unordered Lists

    Unordered lists are lists that are in order that doesn't allow numbers to accompany them.


    Code for unordered list.
    <ul>
    <li> (List item) </li>
    </ul>
    These make lists that look like this:
    • CD
    • Magazine

    Ordered Lists

    Ordered lists elements are <ol> </ol>. Same as unordered lists, but displays the apropriate number next to each list item.

    Such as:

    1. CD
    2. Magazine

    Tables:

    Creating tables for HTML
    <table> </table> is for creating tables with apropriate lists.
    1. First set up a column with <tr> </tr>
    2. <th> begins and ends. Also acts as a heading of each column
    3. <td> </td> is the data in each column under each heading.

    In the end the code for a table looks like this:
    <table>
    <tr>
    <th> (insert column heading) <th>
    <th (Insert column heading #2) <th>
    </tr>
    <tr>
    <td> (Insert data in first column) </td>
    <td>(Insert data in second column) </td>
    </tr>
    <tr>
    <td> (second row of first column) </td>
    <td> (second row of second column) </td>
    </tr>
    </table>

    Links:

    Create redirect to links:

    You need to begin the HTML with <p> </p> or<br>.


    <a href="(url of the link)"> (Then type what you want the name of the imbedded link to be) </a>
    This will be a link imbedded in the text.
    Example:     This is a link of a hot girl
    To create a link that moves the page to a certain spot instead of a different URL.

    Start with the Unordered list and then create list of elements.
    Connect: <ul>
                    <li>   <a href="#(ID or word that moves the page down)"> (same word you want to move it) </a> </li>
                   </ul>
    With:      <a name="(Same ID as the #)" </a>
    # is what connects the links together and when you click the word, it moves the page down.

    Images:

    Adding an image to your HTML file

    You first need the image on the same folder as the HTML file.


    You would need <br> or <p> </p> first to facilitate the text.
    The HTML code would be: <img src="(name of image file)">
    Example:
    <br>
    <img src="(cats.jpg)">

    Resizing and changing images

    Landscapes or big pictures should always use jpeg. Small logos and small portraits should use .png or .gif.

    You simply resize images by altering the image HTML!

    You change <img src="(name of image)"> to <img src="(name of image)" width="(number of pixels)" height="(number of pixels)">
