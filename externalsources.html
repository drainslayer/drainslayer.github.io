The iFrame Element

The iframe element is just another HTML code. It inserts content from another HTML document and inputs in within the current page. Useful for advertisements and music playlists

<iframe frameborder="(1 or 0)" width="(any amount of pixels)" height="(any amoount of pixels)" src="(file in same folder)" name="(give any name to identify)" id="(same as source)"> (any text to click on) </iframe> <br>

To add the second page... add this as well: <a href="(name of second html" target="(same id)"> Some text </a>

This can be itself and doesn't require <form action.
1 means ON. 0 means OFF.
All of these files must be on the same folder.
