First of all, you should always begin with !doctype html!

  1. Start with <!doctype html>. This is to declare what document it is.
  2. Then wrap around our HTML document with <html> , and closing with </html>.
  3. Inside the <html> , there will be <head> </head> and <body> </body>.
  4. The layout of an HTML file goes like this:
    <!doctype html>
    <html>
    <html> <Head>
    <title> or any meta elements
    </head> <body>
    (Any body element)
    </body>
    <html>

Head Breaks

Inside <head></head>, there will always be a title or meta element. <title> </title> is where you can do the name of the tabs of the HTML file you have created.


Body Breaks

  • Inside <body> </body>, there will be multiple things you can add. The <body> is where you can display the actual items on a website.

Small elements

  • <h1> </h1> to <h6> </h6> is the heading. The higher the number the smaller the font of the heading. This can act as a title for the website.
  • <p> </> is for paragraphs or to skip a line within text.
  • <br> are for skipping lines. Does not need to end with a closing element such as </br>. You can add multiple breaks and don't need to end.
  • <!-- (insert comment) --> are for invisible comments.
  • To comment in the <br> or empty spaces, just type next to or beneath the html code.
  • &nbsp is for a space between words or letters. For example: Skip zero ==> skip     zero.
